# MultiChatPDF Query Application

This application allows you to query multiple PDF files simultaneously using the Gemini search engine. It provides a convenient way to search for specific keywords or phrases across multiple PDF documents.

## Prerequisites

Before using this application, make sure you have the following:

- Python=3.10 installed on your machine
- The Gemini gem installed (`gem install gemini`)
- A valid `.env` file containing the necessary configuration variables (see below)
